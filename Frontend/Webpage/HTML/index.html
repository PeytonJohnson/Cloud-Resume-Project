Recent Projects

Cloud Resume

This website is an end product of the cloud resume challenge. The project challenges participants to produce a website to host participants' portfolios running on serverless cloud infrastructure while building a CI/CD pipeline for the website. The challenge requires a visitor counter. I overcome this part of the challenge by using js to call API GateWay, which then triggers a Lambda function. The Lambda function then gets, updates, and adds plus one to the visitor counter stored in a DynamoDB table. Github actions provide continuous development to the website. View the code needed for this project on my GitHub!
